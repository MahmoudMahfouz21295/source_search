# source_search

This is a simple Python script that allows me to find interesting data in the source code of any web page.

Data Example:
 - Comments
 - Javascript Files
 - Meta-Tags Information
 
Usage :-
  python3 source_search.py https://www.domain.com/
  
  
Requirements:
- requests library
- urllib3 library
- re library
- colorado library
